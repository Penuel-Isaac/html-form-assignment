# html-form-assignment
```markdown
Enhanced HTML5 Form Project

Description
This project demonstrates the use of advanced HTML5 elements, including lists, tables, media, and form features. It showcases best practices in content organization, semantic structure, and client-side form validation without the use of JavaScript.

Features
- Structured content using:
  - Ordered and unordered lists
  - HTML tables
  - Embedded media (images, audio, or video)

- A complete HTML5 form with:
  - Text input, email, number, date, checkbox, radio buttons, and select fields
  - Semantic structure using fieldset, label, and legend
  - HTML5 form attributes like required, pattern, minlength, placeholder, readonly, and autocomplete
  - Built-in HTML5 validation for user-friendly data input

Technologies Used
- HTML5
- Semantic HTML elements
- No JavaScript or external libraries used

How to Use
1. Clone the repository or download the enhanced-form.html file.
2. Open it in any modern web browser.
3. Interact with the form and see the built-in validation in action.
